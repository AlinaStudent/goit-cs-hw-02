Инструкция по запуску скрипта website_status_checker.sh в Ubuntu
=============================================================

1. Открой терминал
   Нажмите Ctrl + Alt + T или найдите "Терминал" в меню приложений.

2. Перейдите в папку, куда вы скачали файл
   Например, если файл находится в папке "Загрузки":
   cd ~/Загрузки

3. Проверьте, что файл существует
   ls website_status_checker.sh

4. Сделайте файл исполняемым
   chmod +x website_status_checker.sh

5. Запустите скрипт
   ./website_status_checker.sh

6. Просмотрите результаты
   Скрипт выведет результат в терминал и создаст файл лога с результатами:
   website_status.log

7. Чтобы просмотреть содержимое лога, выполните:
   cat website_status.log

Пример вывода:
<https://google.com> is UP
<https://facebook.com> is UP
<https://twitter.com> is UP
Результати записані в файл лога: website_status.log

Поздравляем! Скрипт успешно выполнен.
